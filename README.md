# Temporal Difference Learning: SARSA vs Q-Learning on CliffWalking

## Overview
This Jupyter notebook lab focuses on comparing two popular reinforcement learning algorithms: SARSA (State-Action-Reward-State-Action) and Q-Learning. Both algorithms will be evaluated in the context of the CliffWalking environment, which presents a unique challenge for learning optimal policies.

## Objective
The main objective of this lab is to demonstrate the differences in the learning strategies of SARSA and Q-Learning and to evaluate their performance in the CliffWalking task.

## Setup Instructions
To set up the environment for running the Jupyter notebook, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/rvallaboju/MSDS684_Week-4--Lab-4.git
   cd MSDS684_Week-4--Lab-4
   ```

2. **Create a virtual environment** (optional but recommended):
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows use `venv\Scripts\activate`
   ```

3. **Install required dependencies**:
   Make sure you have all the required libraries installed:
   ```bash
   pip install -r requirements.txt
   ```

The following packages are required to run the notebook:
